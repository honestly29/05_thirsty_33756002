# The Thirsty Student – Web App

A simple Node.js and Express web application built in the Dynamic Web Applications lab to demonstrate dynamic routing, EJS templating, and basic form handling.

---

## Overview

**The Thirsty Student** is a fictional drinks shop website that:

- Displays shop information and locations.
- Lists product categories.
- Includes a customer **survey page** that collects user input (first name, surname, email, age, favourite drink category, and whether they are a student).
- Displays the survey results on a new page after submission.

No registration or database storage is required — survey results are only shown temporarily on screen.

---

## Technologies Used

- **Node.js** with **Express**
- **EJS** (Embedded JavaScript templates)
- **HTML5 / CSS3**
- **JavaScript**

---

## Installation & Running Locally

1. **Install dependencies**
   ```bash
   npm install express ejs
   ```
2. **Run the app**
   ```
   node index.js
   ```
3. **Open in browser**
   http://localhost:8000/
